How To Play the Melody Game

Object of the Game

Correctly repeat a sequence of colors.


Instructions

  1. Turn on the game, and select the level and the number of rounds you would like to play. Level 1 is the slowest, and level 3 is the fastest.
  2. Press start to begin playing.
  3. Simon will display the first color.
  4. Repeat the color by pressing the same color.
  5. Simon will repeat the color and add a new color on each round.
  6. Continue to repeat the colors in the correct sequence.
  7. If you press an incorrect color in the sequence, Melody will buzz an error sound, and repeat the sequence again if strict mode has not been selected. If strict mode has been selected the game will end on an incorrect color press, and will restart with a new sequence.
  8. The game will continue until the rounds selected are over.
  9. The game speeds up as the sequence grows.

How to Win

To win the game, repeat the sequence of colors in the correct order, until the number of rounds are over.
